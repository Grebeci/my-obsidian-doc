HTML 结构

```html
<!doctype html>
<html lang="zh-CN">
  <head>
    <meta charset="utf-8" />
    <title>我的测试站点</title>
  </head>
  <body>
    <p>这是我的页面</p>
  </body>
</html>
```

1. `<!DOCTYPE html>`: 声明文档类型。这是历史遗留标签，这种写法是最简的。
2. `<html></html>` ： 根元素，包含页面的所有内容。
3.  `<meta charset="utf-8">` ：这个元素代表了不能由其他 HTML 元相关元素表示的元数据， follow 这个即可。
