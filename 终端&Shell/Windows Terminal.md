#### 技巧




终端必备功能

- 选择： 拖动鼠标选择，点到点选择；块选择（多光标）；
- 复制，粘贴简单（尽量不改变快捷键）
- 搜索， 支持从文本缓冲区搜索。
- 支持 Virtual terminal sequence
- 支持自定义主题，颜值党
- 支持多标签
