# 快捷键

#### Mac

| 窗口管理                    | 功能           |     |
| ----------------------- | ------------ | --- |
| Commnad + space         | 全局搜索         |     |
| `Command (⌘) + W`       | **关闭当前窗口**：  |     |
| `Command (⌘) + Q`       | **退出当前应用**：  |     |
| `Command (⌘) + M`       | **最小化当前窗口**： |     |
| `Command (⌘) + H`       | **隐藏当前应用**：  |     |
| `Control + Command + F` | 退出最大化        |     |
