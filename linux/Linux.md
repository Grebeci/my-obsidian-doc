command option args

###### POSIX参数风格

- 短选项： 一个 `-` 引领一个英文字母，多个短选项可以合在一起，其后参数可以不加空格

- 长选项： `--` 引领一个英文单词

- 长选项选项可以携带变量，可用短选项改写

	```shell
	date --set="1900-01-01 08::00:00"
	date -s "1900-01-01 08::00:00"
	```

- 空格敏感，参数信息包含空格，需要用引号括起来。
