**Bash 行操作**

> [行操作 - 《阮一峰 Bash 脚本教程》 - 书栈网 · BookStack](https://www.bookstack.cn/read/bash-tutorial/docs-readline.md)

- `Ctrl + a`：移到行首
    
- `Ctrl + e`：移到行尾。
    
- `Alt + f`：移动到当前单词的词尾。
    
- `Alt + f`：移动到当前单词的词尾。
    
- `Ctrl + w`：删除光标前面的单词。
    
- `Ctrl + k`：剪切光标位置到行尾的文本。
    
- `Ctrl + u`：剪切光标位置到行首的文本。
    
- `Alt + d`：剪切光标位置到词尾的文本。
    
- `Alt + Backspace`：剪切光标位置到词首的文本。
    
- `Ctrl + y`：在光标位置粘贴文本。
    

##### 补全

- Tab：完成自动补全。
    
- `Alt + $`：变量名补全。
    
- `Alt + Tab`：尝试用`.bash_history`里面以前执行命令，进行补全。
