2. Focus(切焦)

  | Focus          | 快捷键                            |
  | -------------- | -------------------------------- |
  | Editor         | `Esc`                            |
  | Navigation bar | Alt+Home                         |
  | Tool windows   | project `Alt+1`                  |
  |                | Terminal `Alt+F12`               |
  | Popup menus    | Generate  `Alt+Insert`           |
  |                | Refactor This `Ctrl+Alt+Shift+T` |
  |                | Project New  `Alt+Insert`        |
  |                |                                  |
